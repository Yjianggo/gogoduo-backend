# 微信小程序考试答题系统

一个基于微信云开发的免费在线考试系统，支持扫码登录、输入信息登录、在线答题和成绩查询。

## 功能特性

- 📱 **扫码登录** - 支持扫描考场二维码快速进入考试
- 👤 **信息验证** - 学生输入学号/工号和姓名进行身份验证
- ⏱️ **倒计时考试** - 30分钟限时考试，实时显示剩余时间
- 📝 **随机抽题** - 每次考试从题库中随机抽取20道题目
- 📊 **成绩统计** - 考试结束立即显示得分和正确率
- 🎖️ **成绩等级** - 自动评定优秀、良好、中等、及格、不及格等级

## 快速开始

### 1. 配置云开发环境

1. 注册微信小程序账号，获取 AppID
2. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
3. 在微信开发者工具中创建新项目，选择「云开发」模板
4. 填写你的 AppID
5. 在云开发控制台中创建环境（如果还没有）

### 2. 更新配置

1. 打开 `project.config.json` 文件
2. 将 `appid` 替换为你的小程序 AppID
3. 将 `envId` 替换为你的云开发环境 ID

```json
{
  "appid": "你的AppID",
  "envId": "你的环境ID"
}
```

### 3. 创建云数据库集合

在微信开发者工具中打开「云开发控制台」，创建以下集合：

1. **exam_users** - 考生信息表
2. **exam_sessions** - 考试会话表
3. **exam_records** - 考试记录表
4. **questions** - 题目表

### 4. 导入题目数据

有两种方式添加题目：

**方式一：手动添加**
在云开发控制台的 questions 集合中手动添加题目，格式如下：

```json
{
  "content": "题目内容",
  "optionA": "A选项内容",
  "optionB": "B选项内容",
  "optionC": "C选项内容",
  "optionD": "D选项内容",
  "correctAnswer": "B",
  "difficulty": 1
}
```

**方式二：使用默认题目**
系统内置了20道默认题目，当题库为空时自动使用。

### 5. 部署云函数

在微信开发者工具中：

1. 右键点击 `cloudfunctions` 文件夹
2. 选择「上传并部署：云端安装依赖」
3. 依次部署三个云函数：
   - login
   - getQuestions
   - submitAnswer

### 6. 运行项目

1. 点击「编译」按钮
2. 在模拟器中查看效果
3. 使用微信扫描二维码进入考试

## 使用说明

### 考生操作流程

1. **扫码入口**：打开小程序首页，点击「扫码进入考场」
2. **手动登录**：也可以点击「手动输入登录」直接进入登录页面
3. **填写信息**：输入学号/工号和姓名
4. **同意须知**：阅读并勾选考试须知
5. **开始考试**：点击「进入考试」开始答题
6. **答题过程**：
   - 每道题有ABCD四个选项
   - 点击选项即可选择答案
   - 可以随时切换题目
   - 底部显示题目导航
7. **提交试卷**：答完所有题目后点击「交卷」
8. **查看成绩**：提交后显示得分和成绩等级

### 考试时间

- 默认考试时间：30分钟
- 时间结束自动交卷

### 评分规则

- 每题5分
- 总分100分（20题 × 5分）
- 90分以上：优秀
- 80-89分：良好
- 70-79分：中等
- 60-69分：及格
- 60分以下：不及格

## 技术栈

- **前端**：微信小程序 WXML + WXSS + JavaScript
- **后端**：微信云开发（云函数 + 云数据库）
- **数据库**：云数据库 MongoDB

## 注意事项

1. 请确保云开发环境已开通并正确配置
2. 云函数需要部署后才能正常使用
3. 题库为空时系统会使用默认题目
4. 考试时间结束会自动交卷

## 常见问题

**Q: 为什么扫码后没有反应？**
A: 请检查是否在首页直接扫码，扫码需要在微信中长按识别小程序码。

**Q: 题目为什么不显示？**
A: 请检查云函数是否部署成功，以及 questions 集合是否有数据。

**Q: 如何添加更多题目？**
A: 在云开发控制台中手动添加，或调用数据库 API 批量导入。
