## 关键技术约定

- 云函数每个目录下需要有 `config.json`，否则开发者工具无法上传
- 微信云数据库"upsert"写法：先 `update()`，失败再 `add()`，不能用 `doc().get()` 判断存在性（文档不存在时会抛异常）
- 云端读取成功后必须同步到 `wx.setStorageSync`，防止云函数偶尔失败时降级读到旧数据
